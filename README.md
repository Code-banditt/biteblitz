BiteBlitz is a full-stack pizza ordering web application built with **Next.js**, **Tailwind CSS**, and **MongoDB**. It provides a smooth, interactive, and responsive experience for users to browse the menu, place orders, and track them in real-time.

## Features

- User authentication with **NextAuth** (email/password + Google Sign-In)
- Browse pizzas, add items to the cart, and place orders
- Confirm orders and track delivery status
- Responsive design with mobile-first layouts
- Interactive animations with **Framer Motion**
- Admin CRUD for pizzas and orders

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API routes, MongoDB, Mongoose
- **Auth:** NextAuth
- **Animations:** Framer Motion
- **Deployment Ready:** Optimized for production build
